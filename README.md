# rhotacism_classification

### Features
- [x] Record Audio
- [x] Detection Rhotacism

### Install library
```
pip install requirements.txt
```

### Run program
```
python test.py --file "cv-invalid/Perekaman baru 125.wav" 
```

### Run For Raspberry
```
source venv/bin/activate
cd my_project
python main.py
```

### Check USB Device
```
lsusb
```

### Run Feature Extraction
```
python mfcc.py
```
